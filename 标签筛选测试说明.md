# 标签筛选功能测试说明

## 如何看到标签筛选功能

### 前提条件
1. **必须已登录**
2. **必须有爱豆**
3. **必须有写过日记，并且日记添加了标签**

### 查看步骤

#### 1. 在微信开发者工具中刷新
   - 点击工具栏的"编译"按钮
   - 或者按 `Ctrl + B` 重新编译

#### 2. 确保有带标签的日记
   - 点击"+"按钮写一条新日记
   - 在"追星标签"区域选择标签（如：演唱会、生日、打歌等）
   - 保存日记
   - 再写几条日记，选择不同的标签

#### 3. 查看标签筛选栏
   - 回到日记列表（底部导航"日记"标签）
   - 在页面顶部"时光日记"标题下方
   - 应该能看到 "🏷️ 按标签筛选" 和横向滚动的标签列表

#### 4. 测试筛选功能
   - 点击任意标签，只显示包含该标签的日记
   - 再次点击标签或点击"清除"按钮，恢复显示所有日记
   - 每条日记卡片内容下方也会显示标签

## 如果还是看不到

### 排查步骤

1. **检查是否有日记**
   - 日记列表是否为空？
   - 如果为空，先写几条日记

2. **检查日记是否有标签**
   - 打开开发者工具的调试器（AppData）
   - 查看 `pages/diary/diary` 页面的 `data.allTags`
   - 如果是空数组 `[]`，说明你的日记没有标签

3. **如何给现有日记添加标签**
   - 点击日记右上角的"..."菜单
   - 选择"编辑日记"
   - 在"追星标签"区域选择标签
   - 保存

4. **清除缓存重试**
   - 微信开发者工具 → 工具 → 清除缓存 → 清除全部缓存
   - 重新编译运行

## 测试场景

### 场景1：写新日记测试
1. 点击"+"写日记
2. 选择"演唱会"标签
3. 保存
4. 再写一条，选择"生日"标签
5. 再写一条，选择"演唱会"和"打歌"标签
6. 回到日记列表，应该看到3个标签按钮

### 场景2：筛选测试
1. 点击"演唱会"标签
2. 应该只显示2条日记（有"演唱会"标签的）
3. 点击"清除"或再次点击"演唱会"
4. 恢复显示所有3条日记
